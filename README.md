Shibafu
===

自分専用TODOアプリ。はじめてのiOSアプリ挑戦。

![1](media/Shibafu_1.jpg)

![2](media/Shibafu_2.jpg)

## Description

- ふつーのTODO機能
- タスクはプレーンテキストでDropbox上に置かれる
- ぱそこのエディタでも、アプリ上でも編集可能
- タスクをこなし模様をGitHub的芝生で閲覧

## Installation

```bash
cd shibafu
brew install cocoapods
pod setup
pod install
```

あと、 xcode の環境設定から自分の Apple Account を登録。
